Bienvenida

Bienvenido a mi sitio web personal, desarrollado como parte del trabajo práctico de la asignatura Fundamentos de Ingeniería Informática.

Este sitio tiene como objetivo mostrar los conocimientos adquiridos en HTML y CSS, así como documentar mi progreso académico a lo largo del Grado en Ingeniería Informática.

Estructura del sitio

  • Sobre mí: información personal y académica.
  • Grado: descripción del grado y asignaturas.
  • Tema: desarrollo de un tema de interés.
  • Red: enlaces a páginas web de compañeros.
  • Contacto: formulario de contacto.
