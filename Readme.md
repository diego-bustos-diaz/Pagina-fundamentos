# Sitio web personal – Diego Bustos

## Descripción del trabajo

Este repositorio contiene el desarrollo de un sitio web personal realizado
como parte del trabajo práctico de la asignatura Fundamentos de Ingeniería
Informática del Grado en Ingeniería Informática en la Universidad Francisco de
Vitoria.

El objetivo principal del proyecto es aplicar los conocimientos adquiridos
sobre HTML y CSS para crear un sitio web estructurado, funcional y visualmente
coherente, que refleje el progreso académico y los intereses del estudiante.

El sitio web incluye diferentes secciones, entre las que se encuentran:
- Página principal (index.html)
- Información personal (about.html)
- Descripción del grado (degree.html)
- Página dedicada a Fundamentos de Ingeniería Informática (fii.html)
- Desarrollo de un tema de interés (topic.html)
- Formulario de contacto (contact.html)
- Red de compañeros (net.html)

## Problemas durante el desarrollo

Durante el desarrollo del proyecto se presentaron diversas dificultades,
principalmente relacionadas con la organización de la estructura de
carpetas, el uso adecuado de rutas relativas entre los distintos archivos HTML
y la aplicación de estilos CSS de forma común a todas las páginas.

## Conclusiones

La realización de este proyecto ha permitido aprender los conocimientos
básicos sobre el desarrollo web mediante HTML y CSS, así como comprender la
importancia de una buena organización del código y del control de versiones
mediante Git y GitHub.

Además, el proyecto ha servido como una introducción práctica al diseño de
sitios web.
